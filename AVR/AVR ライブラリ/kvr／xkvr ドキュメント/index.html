KVR / XKVR

AVR ATtiny / ATmega / ATxmega Library

概要

KVR / XKVR は、Atmel 社の AVR マイコン向けに書かれた C 言語のライブラリです。
利用には、別途 avr-libc が必要です (WinAVR には既に含まれています)。

動作環境

* ライブラリによっては、対応していないマイコンも存在します。
