kvr/base.h

/ > kvr/base.h

概要

各ライブラリのベースになるライブラリです。
マイコンの種類による分岐を行うためのマクロの定義 (#define) を行います。

サンプル

#define F_CPU 8000000 // 8 MHz

#include <avr/io.h>
#include <util/delay.h>

#include "kvr/base.h"

int main(){
	
#ifdef KVR_BASE_XX8
	// ATmega88p, ATmega168p, 他 28 ピンマイコン
#endif

#ifdef KVR_BASE_XX4
	// ATmega644p, ATmega1284p, 他 40 ピンマイコン
#endif
	
    return 0;
}

リファレンス

KVR_BASE_XX4

以下のマイコンで定義されます。

  • ATmega164P
  • ATmega164A
  • ATmega324P
  • ATmega324A
  • ATmega324PA
  • ATmega644
  • ATmega644P
  • ATmega644A
  • ATmega644PA
  • ATmega1284P

KVR_BASE_XX8

以下のマイコンで定義されます。

  • ATmega48
  • ATmega48A
  • ATmega48P
  • ATmega88
  • ATmega88A
  • ATmega88P
  • ATmega88PA
  • ATmega168
  • ATmega168A
  • ATmega168P
  • ATmega328
  • ATmega328P

KVR_BASE_2313

以下のマイコンで定義されます。

  • ATtiny2313
  • ATtiny2313A
