kvr/i2c/master.h

/ > kvr/i2c/master.h

概要

I2C によるシリアル通信を行うライブラリです。
I2C には、マスター(主装置) とスレーブ (従装置) があり、マスターとして動作する場合の実装です。

サンプル

#define F_CPU 20000000UL // 20 MHz

#include <avr/io.h>
#include <util/delay.h>

#include "kvr/i2c/master.h"

int main(void){
	
    return 0;
}

リファレンス

初期化

// I2C マスター 初期化
bool i2c_master_init(
	const uint32_t f // SCL 周波数
	);
  • f: SCL 周波数 (Hz; 最大 400 kHz)

I2C マスターの初期化を行います。
プログラムの最初で、一度だけ呼び出してください。

標準入出力設定

// 標準出力に設定
void usart_stdout(
	volatile const uint8_t num
	);

// 標準入力に設定
void usart_stdin(
	volatile const uint8_t num
	);

// 標準入出力
void usart_stdio(
	volatile const uint8_t num
	);
  • num: USART 番号

標準入出力に設定すると、printf, scanf 等の標準入出力関数が使用できるようになります。
ただし、プログラムサイズの関係上、この機能は ATtiny2313 では使用できません。

文字出力

void usart_send(
	volatile uint8_t num,
	volatile char c
	);

void usart_putchar(
	volatile const uint8_t num,
	volatile const int c
	);
  • num: USART 番号
  • c : 出力する文字

文字を出力します。

文字列出力

void usart_print(
	volatile const uint8_t num,
	const char *s
	);
	
void usart_puts(
	volatile const uint8_t num,
	const char *s
	);
  • num: USART 番号
  • s : 出力する文字列

文字列を出力します。
usart_puts 関数は、文字列を出力後改行 (LF) を行います。

整数出力

void usart_putint(
	volatile const uint8_t  num,
	volatile       uint32_t n
	);
  • num: USART 番号
  • n : 出力する整数

整数を出力します。
32 bit 以内の整数であれば、他の型でも出力できます (long, int, char, uint8_t, etc ...)。

文字入力

uint8_t usart_receive(
	volatile const uint8_t num
	);

uint8_t usart_getchar(
	volatile const uint8_t num
	);

文字の入力を行います。
受信するまでブロック (待機) します。
