kvr/io/bitfield.h

/ > kvr/io/bitfield.h

概要

入出力ビットフィールドライブラリ。

サンプル

#define F_CPU 8000000 // 8 MHz

#include <avr/io.h>
#include <util/delay.h>

#include "kvr/io/bitfield.h"

int main(void){
    
    while(1){
        DDRC_0 = 1;
        DDRC_2 = 1;
        
        while(1){
            PORTC_0 = 1;
            PORTC_2 = 0;
        }
    }
    
    return 0;
}

リファレンス

該当するマイコンに存在するならば、以下のマクロが定義されます。

  • PORTA_0 - PORTA_7
  • PORTB_0 - PORTB_7
  • PORTC_0 - PORTC_7
  • PORTD_0 - PORTD_7
  • PINA_0 - PINA_7
  • PINB_0 - PINB_7
  • PINC_0 - PINC_7
  • PIND_0 - PIND_7
  • DDRA_0 - DDRA_7
  • DDRB_0 - DDRB_7
  • DDRC_0 - DDRC_7
  • DDRD_0 - DDRD_7
