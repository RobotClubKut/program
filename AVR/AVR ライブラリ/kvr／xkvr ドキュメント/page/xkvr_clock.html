概要

ATxmega では、プログラムからマイコンの動作クロックを設定します。
このライブラリを使うと、動作クロックの設定を簡単に行うことができます。

サンプル

#include "xkvr/clock.h"

int main(){
    
    // 内部クロック 32 MHz に設定
    clock_internal();
    
    // 外部クロック 32 MHz に設定
    // 8 MHz の発振子を PLL にて 4 倍に逓倍 (8 MHz x 4 = 32 MHz)
//  clock_external();
    
    while(1){
        ...
    }
		
    return 0;
}
